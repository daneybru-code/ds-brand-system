# Serviços

A DS Design & Motion oferece quatro pilares de serviço para a comunicação visual de eventos.

## Design para Eventos

Criação de peças visuais para divulgação e organização de eventos: save the date, cards para feed, stories, banners digitais, artes para WhatsApp, programação visual, peças institucionais, telas de abertura e templates para divulgação.

## Impressos para Eventos

Materiais físicos para uso no evento: crachás, certificados, fichas de palco, backdrop, banners, totens, sinalização, credenciais e programação impressa.

## Motion Design

Animações e vídeos curtos para aumentar percepção de valor e impacto visual: vinhetas, chamadas animadas, lettering animado, lower thirds, abertura de evento, telas animadas para telão e versões animadas de posts.

## Edição de Vídeos

Edição e finalização de vídeos curtos para divulgação: corte de vídeo, organização de ritmo, legendas, trilha, inserção de identidade visual e adaptação para Reels, Stories e WhatsApp.
