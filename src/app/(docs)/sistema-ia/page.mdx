# Sistema de IA da Agência

A IA é usada como apoio estratégico e operacional. Ela não substitui o olhar criativo, mas acelera organização, análise e documentação.

## Funções da IA

- Organizar briefing
- Resumir reuniões e conversas
- Criar caminhos criativos
- Gerar ideias de títulos e chamadas
- Criar legendas
- Montar propostas
- Criar checklists
- Revisar escopo
- Sugerir cronogramas
- Criar roteiros para vídeos
- Ajudar na análise de público
