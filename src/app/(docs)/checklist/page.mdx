# Checklist de Produção

## Antes de começar

- Briefing completo recebido
- Logos recebidas
- Textos obrigatórios recebidos
- Fotos ou imagens recebidas
- Formatos definidos
- Prazo confirmado
- Valor aprovado
- Entrada paga
- Pasta do projeto criada

## Durante a criação

- Criar direção visual
- Definir paleta
- Definir hierarquia de informações
- Criar primeira peça base
- Adaptar peças derivadas
- Conferir ortografia
- Conferir datas, horários e locais
- Conferir logos
- Conferir margens e sangria, se impresso
- Exportar prévias

## Revisão

- Enviar prévia ao cliente
- Registrar ajustes solicitados
- Conferir se ajustes estão dentro do escopo
- Aplicar ajustes aprovados
- Enviar versão final para aprovação

## Entrega

- Exportar JPG/PNG para redes
- Exportar PDF para impressão, se houver
- Exportar MP4, se houver motion/vídeo
- Organizar arquivos por formato
- Enviar link final
- Registrar projeto no controle interno
- Solicitar feedback
- Perguntar sobre uso no portfólio
