# Organização de Arquivos

## Estrutura de pastas

```
DS DESIGN & MOTION/
├── 01_CLIENTES/
│   └── Nome_do_Cliente/
│       └── Nome_do_Projeto/
│           ├── 01_BRIEFING/
│           ├── 02_REFERENCIAS/
│           ├── 03_ARQUIVOS_RECEBIDOS/
│           ├── 04_DESIGN/
│           ├── 05_MOTION_VIDEO/
│           ├── 06_PREVIAS/
│           ├── 07_EXPORTS_FINAIS/
│           └── 08_PROPOSTA_CONTRATO/
├── 02_MODELOS/
├── 03_PORTFOLIO/
├── 04_FINANCEIRO/
└── 05_BRAND_SYSTEM/
```

## Padrão de nome dos arquivos

```
cliente_projeto_peca_formato_v01
cliente_projeto_peca_formato_v02
cliente_projeto_peca_formato_FINAL
```

**Exemplo:**

```
lukaplan_endogineco_post_feed_v01.png
lukaplan_endogineco_story_palestrante_v02.png
lukaplan_endogineco_certificado_FINAL.pdf
```
