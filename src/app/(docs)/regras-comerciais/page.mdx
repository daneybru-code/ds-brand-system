# Regras Comerciais

## Pagamento

- Projetos avulsos: 50% para iniciar e 50% antes da entrega final.
- Projetos mensais: pagamento no início do ciclo.
- Demandas urgentes podem ter taxa adicional.

## Revisões

- Cada pacote possui limite de revisões.
- Revisão é ajuste dentro do caminho aprovado.
- Mudança de briefing é novo orçamento.

## Prazos

- O prazo começa após pagamento de entrada e envio completo dos materiais.
- Atraso no envio de textos, logos ou aprovações pode alterar a data de entrega.

## Arquivos abertos

- Arquivos editáveis não estão inclusos por padrão.
- Caso o cliente solicite arquivos abertos, isso deve ser orçado separadamente.

## Impressão

- A criação de arquivos para impressão pode estar inclusa no pacote.
- A impressão física não está inclusa, salvo negociação específica.

## Política de Escopo

**Tudo que não estiver descrito na proposta não está incluso no projeto.**

O que precisa estar claro antes de começar:

- Quantidade de peças
- Formatos
- Prazos
- Canais
- Quantidade de revisões
- Materiais fornecidos pelo cliente
- Responsável pela aprovação
- Responsável pelos textos
- Responsável pelas logos
- Se haverá motion ou não
- Se haverá impressão ou não
