# Scripts de Atendimento

## Primeiro contato

> Olá! Obrigado pelo contato. Para eu entender melhor o projeto e te passar uma proposta justa, preciso saber quais peças você precisa, qual o objetivo da divulgação e qual o prazo de entrega. Posso te enviar algumas perguntas rápidas de briefing?

## Quando o cliente pede preço sem briefing

> Consigo te passar uma estimativa, mas para fechar um valor correto preciso entender o escopo. Uma arte avulsa, um pacote de divulgação e uma comunicação completa de evento têm níveis bem diferentes de trabalho. Me manda as informações principais que eu organizo uma proposta para você.

## Quando o cliente diz "é só uma arte simples"

> Entendo. Mesmo uma peça simples precisa de objetivo, texto correto, formato, hierarquia visual e adaptação para o canal certo. Para evitar retrabalho, vou organizar o escopo antes e te passo prazo e valor certinho.

## Quando o cliente pede desconto

> Consigo ajustar o escopo para chegar em um investimento mais confortável. Em vez de reduzir apenas o valor, podemos reduzir quantidade de peças, revisões ou complexidade da entrega. Assim o projeto continua justo para os dois lados.

## Quando o cliente pede urgência

> Consigo avaliar a possibilidade de encaixe. Como demanda urgente exige reorganizar agenda, posso te passar uma condição com taxa de urgência ou uma versão reduzida do escopo para atender o prazo.

## Quando o cliente pede revisão fora do escopo

> Esse ajuste altera o escopo aprovado inicialmente. Posso fazer, sem problema, mas preciso orçar como ajuste extra para manter o projeto organizado e justo.

## Pedido de aprovação final

> Segue a versão final para aprovação. Peço que confira com atenção nomes, datas, horários, local, logos e informações obrigatórias. Após aprovação, farei a exportação dos arquivos finais nos formatos combinados.

## Pós-venda

> Projeto entregue! Obrigado pela confiança. Se fizer sentido para você, posso também organizar as próximas peças do evento ou criar uma versão animada para aumentar o impacto da divulgação.
